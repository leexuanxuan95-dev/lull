# Lull · AI Sleep Stories Made For You

> A bedtime story written for who you are tonight.

Lull is an iOS app that generates a personalized sleep story every night —
your name, your city, the slow thing you used to love — entirely on-device.
No LLM call, no cloud. The story engine is a templated grammar that produces
more than 10⁴⁰ unique stories per genre. The Companion chat replies are
generated by a local rule-based grammar (more than 10⁹ unique replies).

This repo has both:
- **`Lull/`** — the iOS app (SwiftUI 6, iOS 17+)
- **`docs/`** — the marketing + privacy/terms/support site, served via GitHub Pages

## Getting started

```sh
brew install xcodegen          # one-time
xcodegen generate              # generates Lull.xcodeproj from project.yml
open Lull.xcodeproj
```

Then run on iPhone simulator (iOS 17+).

## Architecture

| Module | Responsibility |
|---|---|
| `Lull/Engine/StoryGenerator.swift` | Builds a story from `(genre, profile, seed)` |
| `Lull/Engine/StoryGrammar.swift` | Lexical pools + per-genre scene templates |
| `Lull/Engine/CompanionChat.swift` | Façade for the chat reply engine |
| `Lull/Engine/MessageAnalyzer.swift` | Crude intent + topic detector |
| `Lull/Engine/ChatGrammar.swift` | Per-intent reply template pools |
| `Lull/Engine/SeededRandom.swift` | SplitMix64 PRNG + stable hasher |
| `Lull/Engine/VoiceEngine.swift` | AVSpeechSynthesizer wrapper |
| `Lull/Engine/SleepDetector.swift` | Sleep-timer fallback for HealthKit |
| `Lull/Subscription/SubscriptionStore.swift` | StoreKit 2 entitlement bridge |
| `Lull/Persistence/LullVault.swift` | On-disk JSON cache for stories + chat |
| `Lull/Theme/` | Colors, fonts, components, star field |
| `Lull/Views/` | Onboarding · Tonight · Listening · Companion · Voice clone · Smart wake · Paywall · Settings |

## Tests

```sh
xcodebuild test \
  -project Lull.xcodeproj \
  -scheme Lull \
  -destination 'platform=iOS Simulator,name=iPhone 15'
```

The test bundle asserts the combinatoric space exceeds 10⁹ for both stories
and chat replies, and verifies determinism (same seed → same output).
